# Hospital Management System

## Features

- Login
- Patients
- Doctors
- Appointments

## Technologies

- Python
- Streamlit

## How to run

1. Clone the repository
2. Install the dependencies

```bash
pip3 install streamlit
pip3 install mysql-connector-python
pip3 install watchdog
```

Prerequisites: python 3.12 and above

3. Edit the configuration in hospital_management.py
    * Change the database configuration to your own

4. Run the application

```bash
streamlit run streamlit_main.py
```
